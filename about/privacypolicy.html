A “Cookie” is a string of information that a website stores on a user’s computer or mobile device, and that the user’s browser provides to the website each time the user submits a query to the site. ZenoFon uses Cookies to track the pages that users visit during each Website session, both to help ZenoFon improve users’ experiences and to help ZenoFon understand how the Website and services are being used.
